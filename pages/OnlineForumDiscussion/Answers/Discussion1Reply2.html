Information System Failure: 2021 Bitcoin's Flash Crash

According to my research this flash crash was caused from a bug in an institutional traders trading algorithm (Betz, 2021). This is a clear indication that everyone should be testing their application thoroughly, not only positive testing but negative testing as well. Automated tests UI and Unit tests would have helped in this case because automated tests would have been able to run multiple scenarios and mock how a customer might behave. When testing we should always think of ways to break the application because when customers use the application, they do not always use it as intended and they might make mistakes that will bring up an underlying bug in the code which might cause a huge financial loss for others.

References:

Betz, Brandy. “Bitcoin Price Flash Crash on Binance.US Attributed to Trader Algorithm Bug.” Www.coindesk.com, 21 Oct. 2021, www.coindesk.com/markets/2021/10/21/bitcoin-price-flash-crash-on-binanceus-attributed-to-trader-algorithm-bug/. Accessed 11 Feb. 2022.
