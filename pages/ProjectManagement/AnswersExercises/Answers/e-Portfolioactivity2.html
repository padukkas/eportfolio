Question: Use some Python code which you have developed in the past and apply at least 3 of the strategies presented at the source to improve its quality.

Intro

I have used the code that was developed as a group for the final submission for the SSDCS module.

Use tools to help you

I have used VS Code along with PyLint, it has helped me write code quicker as it suggests classes, functions and attributes automatically. Also refactoring code is much easier when using such tools.

Proper structure

The code has been structured in a manner where it is logically structure therefore making it easier to navigate and make quick changes. The containers folders hold everthing related to the project code it self. The main root folder contains the reports, yaml files to perform deployments, readme files and installation files. Making it easier to navigate to locate files and code.

Document all things

The code has been well documented, with instructional step by step guide on getting the application running locally to each tool used and language used to develop the application. Also the code it self is well documented from functions to classes called in the application.

References:

Mertz, J. (n.d.). SQI Seminar Series Writing Clean and Pythonic Code. [online] Available at: https://trs.jpl.nasa.gov/bitstream/handle/2014/51618/CL%2319-5039.pdf?sequence=1.
