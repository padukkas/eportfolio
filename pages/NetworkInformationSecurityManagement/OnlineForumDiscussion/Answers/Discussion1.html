Introduction:

Compromising a Medical Mannequin. Healthcare Information Systems and Technology (Sighealth).

During the research of Compromising a Medical Mannequin they found two components as vulnerable to attacks which includes network security solution and the network protocol. The bad actors were able to breach the security solutions system using an open-source brute force attack against the router Personal identification Number (PIN) and the network protocol was attacked by using a denial-of-service attack. (Glisson et al., n.d.)

There are different actions that we can take to avoid a brute force attack against a Personal identification Number. (O’DRISCOLL, 2018)

  • Using a strong password, strong passwords include longer passphrases, passwords that include combinations like upper case letters, numbers and special characters, never repeating the same password across multiple platforms and using password managers.
  • Limit the number of failed attempts. The application should have validation to block connections that try multiple failed attempts to login to the application.
  • Having security questions, will it more difficult for an attacker to gain access to an application.
  • One of the biggest defenses against a brute force attack is Multi Factor Authentication (MFA) which will require the user’s authorization from a separate application.

    • Threat actors carry out denial of service (DoS) attacks to disrupt the availability of an organization’s services and data. If successful, a DoS attack prevents people from accessing online services (e.g., email, websites, online accounts), information, and other network resources. (Security, 2018)

    There are different methods to avoid a DoS attack. Some of them are, (Dobran, 2019)

  • Deploy firewalls which can detect and mitigate network and application-level attacks.
  • Setup continuous network monitoring systems and applications.
  • Use tools such as IDS (intrusion detection system) and IPS (intrusion prevention system).

    • Above mentioned are just some actions to mitigate future attacks. There are many more other options.
